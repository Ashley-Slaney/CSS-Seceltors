CSS Selectors

To style a web page with CSS, you must use selectors that can manipulate the way HTML looks on a web page.

Before you can style with CSS you must link the stylesheet in the head by using the 'link' tag and setting the path to the CSS file in the href.

A CSS element consists of three things, 'selector', 'property' and 'value'

A CSS selector looks like this:

body {
color : green;
}

The selector in the example above is 'body'. This defines which HTML element to select.

The property in the example above is 'color'. This property defines what is going to happen to the HTML element that has been selected.

The value in the example above is 'green'. This tells the property what action to give to the selected HTML element.
